# Micro App Demo

一个使用[micro-app](https://micro-zoe.github.io/)的微前端demo， 主应用为angular 子应用为React+Vite。

## 运行

```bash
cd angular-main
pnpm start
```

```bash
cd react-vite
pnpm start
```

打开 http://localhost:4200/
